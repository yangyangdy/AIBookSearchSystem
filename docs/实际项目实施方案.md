# 书籍封面向量检索系统 - 实际项目实施方案

## 一、项目实际情况

### 1.1 数据源
- **MySQL表**：`book_info_table`（表名可配置）
- **关键字段**：
  - `id`：主键
  - `sku`：商品SKU
  - `isbn`：ISBN号
  - `cover_link`：图片链接（OSS URL或其他图片链接）
  - 其他业务字段...

### 1.2 技术栈
- **向量数据库**：Milvus（阿里云版）
- **向量化模型**：阿里云 Embedding 模型（通义千问多模态API）
- **OCR服务**：阿里云OCR服务（用于文字提取）
- **数据源**：MySQL（RDS）

### 1.3 核心需求
1. 从 `book_info_table` 分批获取数据
2. 对每行的 `cover_link` 图片进行向量化
3. 同时进行OCR提取文字信息
4. 将向量、OCR文字、id/sku/isbn等信息存入Milvus
5. OCR文字用于相似性查找后的进一步判断

### 2.2 数据流转流程

```
1. 从MySQL分批查询
   ↓
2. 遍历每行数据，获取cover_link
   ↓
3. 下载图片 → 基础验证 → 感知哈希
   ↓
4. 并行处理：
   ├─→ 调用阿里Embedding API → 获得向量
   └─→ 调用阿里OCR API → 获得文字
   ↓
5. 组装数据：向量 + id + sku + isbn + ocr_text + cover_hash
   ↓
6. 批量插入Milvus
   ↓
7. 记录处理进度，支持断点续传
```

from pymilvus import CollectionSchema, FieldSchema, DataType

# 定义字段
fields = [
    FieldSchema(name="id", dtype=DataType.INT64, is_primary=True, description="主键ID"),
    FieldSchema(name="mysql_id", dtype=DataType.INT64, description="MySQL中的ID"),
    FieldSchema(name="sku", dtype=DataType.VARCHAR, max_length=128, description="商品SKU"),
    FieldSchema(name="isbn", dtype=DataType.VARCHAR, max_length=32, description="ISBN号"),
    FieldSchema(name="cover_link", dtype=DataType.VARCHAR, max_length=512, description="图片链接"),
    FieldSchema(name="cover_hash", dtype=DataType.VARCHAR, max_length=16, description="感知哈希"),
    FieldSchema(name="ocr_text", dtype=DataType.VARCHAR, max_length=2048, description="OCR提取的文字"),
    FieldSchema(name="embedding", dtype=DataType.FLOAT_VECTOR, dim=768, description="图片向量")
]

# 创建Schema
schema = CollectionSchema(
    fields=fields,
    description="书籍封面向量检索表"
)

# 创建Collection
collection_name = "book_cover_vectors"

    name=collection_name,
    schema=schema
)

# 创建索引
index_params = {
    "metric_type": "COSINE",

    "params": {

## 四、核心功能实现

### 4.1 批量处理主流程

```python
import pymysql
from pymilvus import connections, Collection
from alibabacloud_dashscope import client
import requests
from PIL import Image
import imagehash
import io
from concurrent.futures import ThreadPoolExecutor, as_completed
import time

class BookCoverVectorProcessor:
    def __init__(self, mysql_config, milvus_config, ali_config):
        """
        初始化处理器
        """
        # MySQL连接
        self.mysql_conn = pymysql.connect(**mysql_config)
        
        # Milvus连接
        connections.connect(**milvus_config)
        self.collection = Collection("book_cover_vectors")
        
        # 阿里云配置
        self.ali_config = ali_config
        client.api_key = ali_config['api_key']
        
        # 配置
        self.table_name = ali_config.get('table_name', 'book_info_table')
        self.batch_size = ali_config.get('batch_size', 1000)
        self.max_workers = ali_config.get('max_workers', 10)
    
    def fetch_batch_from_mysql(self, offset, limit):
        """
        从MySQL分批获取数据
        """
        sql = f"""
        SELECT id, sku, isbn, cover_link 
        FROM {self.table_name}
        WHERE cover_link IS NOT NULL AND cover_link != ''
        LIMIT {limit} OFFSET {offset}
        """
        
        cursor = self.mysql_conn.cursor(pymysql.cursors.DictCursor)
        cursor.execute(sql)
        results = cursor.fetchall()
        cursor.close()
        
        return results
    
    def download_and_validate_image(self, cover_link):
        """
        下载图片并进行基础验证
        """
        try:
            # 下载图片
            response = requests.get(cover_link, timeout=10)
            response.raise_for_status()
            
            # 读取图片
            img = Image.open(io.BytesIO(response.content))
            
            # 格式验证
            if img.format not in ['JPEG', 'PNG', 'WEBP']:
                return None, f"不支持的格式: {img.format}"
            
            # 可读性验证
            img.verify()
            img = Image.open(io.BytesIO(response.content))  # 重新打开
            
            # 计算感知哈希
            cover_hash = str(imagehash.phash(img))
            
            return img, cover_hash
            
        except Exception as e:
            return None, f"图片处理失败: {str(e)}"
    
    def get_embedding(self, image_url):
        """
        调用阿里云Embedding API获取向量
        """
        try:
            response = client.images.embed(
                model='qwen-vl-plus',  # 或使用其他阿里云多模态模型
                input={'image': image_url}
            )
            
            if response.status_code == 200:
                embedding = response.output.embeddings[0]
                return embedding
            else:
                return None
                
        except Exception as e:
            print(f"向量化失败: {e}")
            return None
    
    def get_ocr_text(self, image_url):
        """
        调用阿里云OCR API提取文字
        """
        try:
            # 使用阿里云OCR服务（示例，具体API根据实际服务调整）
            response = client.ocr.recognize(
                image_url=image_url,
                # 其他参数...
            )
            
            if response.status_code == 200:
                # 提取文字内容
                ocr_text = ' '.join([item['text'] for item in response.output.words])
                return ocr_text
            else:
                return ""
                
        except Exception as e:
            print(f"OCR失败: {e}")
            return ""
    
    def process_single_record(self, record):
        """
        处理单条记录
        """
        mysql_id = record['id']
        sku = record.get('sku', '')
        isbn = record.get('isbn', '')
        cover_link = record['cover_link']
        
        # 1. 下载并验证图片
        img, cover_hash = self.download_and_validate_image(cover_link)
        if img is None:
            return None, f"图片验证失败: {cover_hash}"
        
        # 2. 并行处理：向量化 + OCR
        with ThreadPoolExecutor(max_workers=2) as executor:
            embedding_future = executor.submit(self.get_embedding, cover_link)
            ocr_future = executor.submit(self.get_ocr_text, cover_link)
            
            embedding = embedding_future.result()
            ocr_text = ocr_future.result()
        
        if embedding is None:
            return None, "向量化失败"
        
        # 3. 组装数据
        data = {
            'mysql_id': mysql_id,
            'sku': sku,
            'isbn': isbn,
            'cover_link': cover_link,
            'cover_hash': cover_hash,
            'ocr_text': ocr_text,
            'embedding': embedding
        }
        
        return data, None
    
    def process_batch(self, records):
        """
        批量处理记录
        """
        results = []
        errors = []
        
        # 使用线程池并行处理
        with ThreadPoolExecutor(max_workers=self.max_workers) as executor:
            future_to_record = {
                executor.submit(self.process_single_record, record): record 
                for record in records
            }
            
            for future in as_completed(future_to_record):
                record = future_to_record[future]
                try:
                    data, error = future.result()
                    if data:
                        results.append(data)
                    else:
                        errors.append({
                            'record': record,
                            'error': error
                        })
                except Exception as e:
                    errors.append({
                        'record': record,
                        'error': str(e)
                    })
        
        return results, errors
    
    def insert_to_milvus(self, data_list):
        """
        批量插入Milvus
        """
        if not data_list:
            return
        
        # 准备数据（按照schema顺序）
        ids = [i for i in range(len(data_list))]
        mysql_ids = [d['mysql_id'] for d in data_list]
        skus = [d['sku'] for d in data_list]
        isbns = [d['isbn'] for d in data_list]
        cover_links = [d['cover_link'] for d in data_list]
        cover_hashes = [d['cover_hash'] for d in data_list]
        ocr_texts = [d['ocr_text'] for d in data_list]
        embeddings = [d['embedding'] for d in data_list]
        
        data = [
            ids,
            mysql_ids,
            skus,
            isbns,
            cover_links,
            cover_hashes,
            ocr_texts,
            embeddings
        ]
        
        # 插入数据
        self.collection.insert(data)
        self.collection.flush()
    
    def run(self, start_offset=0, max_records=None):
        """
        主处理流程
        """
        offset = start_offset
        total_processed = 0
        total_errors = 0
        
        print(f"开始处理，起始偏移: {offset}")
        
        while True:
            # 1. 从MySQL获取一批数据
            records = self.fetch_batch_from_mysql(offset, self.batch_size)
            
            if not records:
                print("没有更多数据，处理完成")
                break
            
            print(f"获取到 {len(records)} 条记录，开始处理...")
            
            # 2. 批量处理
            results, errors = self.process_batch(records)
            
            # 3. 插入Milvus
            if results:
                self.insert_to_milvus(results)
                print(f"成功处理 {len(results)} 条，插入Milvus")
            
            if errors:
                total_errors += len(errors)
                print(f"处理失败 {len(errors)} 条")
            
            # 4. 更新进度
            offset += len(records)
            total_processed += len(records)
            
            print(f"进度: {total_processed} 条，成功: {len(results)}，失败: {len(errors)}")
            
            # 5. 检查是否达到最大记录数
            if max_records and total_processed >= max_records:
                break
            
            # 6. 短暂休息，避免过载
            time.sleep(0.1)
        
        print(f"处理完成！总计: {total_processed}，成功: {total_processed - total_errors}，失败: {total_errors}")

# 使用示例
if __name__ == "__main__":
    mysql_config = {
        'host': 'your_mysql_host',
        'user': 'your_user',
        'password': 'your_password',
        'database': 'your_database',
        'charset': 'utf8mb4'
    }
    
    milvus_config = {
        'alias': 'default',
        'host': 'your_milvus_host',
        'port': '19530'
    }
    
    ali_config = {
        'api_key': 'your_ali_api_key',
        'table_name': 'book_info_table',  # 可配置
        'batch_size': 1000,
        'max_workers': 10
    }
    
    processor = BookCoverVectorProcessor(mysql_config, milvus_config, ali_config)
    processor.run(start_offset=0)
```

class BookCoverSearchService:
    def __init__(self, milvus_config):
        """
        初始化检索服务
        """
        connections.connect(**milvus_config)
        self.collection = Collection("book_cover_vectors")
        self.collection.load()
    
    def search_by_image(self, image_url, top_k=10, similarity_threshold=0.7):
        """
        通过图片进行相似度检索
        """
        # 1. 对查询图片进行向量化
        embedding = self.get_embedding(image_url)
        if embedding is None:
            return []
        
        # 2. 在Milvus中搜索
        search_params = {
            "metric_type": "COSINE",
            "params": {"ef": 64}
        }
        
        results = self.collection.search(
            data=[embedding],
            anns_field="embedding",
            param=search_params,
            limit=top_k,
            output_fields=["mysql_id", "sku", "isbn", "cover_link", "ocr_text"]
        )
        
        # 3. 过滤相似度阈值
        filtered_results = []
        for hits in results:
            for hit in hits:
                if hit.score >= similarity_threshold:
                    filtered_results.append({
                        'id': hit.id,
                        'mysql_id': hit.entity.get('mysql_id'),
                        'sku': hit.entity.get('sku'),
                        'isbn': hit.entity.get('isbn'),
                        'cover_link': hit.entity.get('cover_link'),
                        'ocr_text': hit.entity.get('ocr_text'),
                        'similarity': hit.score
                    })
        
        return filtered_results
    
    def search_with_ocr_filter(self, image_url, ocr_keywords=None, top_k=10):

## 七、性能优化建议

### 7.1 批量处理优化

1. **分批大小**：根据实际情况调整（建议 1000-10000）
2. **并发控制**：控制线程数，避免API限流
3. **断点续传**：记录处理进度，支持中断恢复
4. **错误处理**：记录失败记录，支持重试

### 7.2 API调用优化

1. **批量API**：如果阿里云支持批量向量化，优先使用
2. **缓存机制**：相同cover_link的图片缓存向量结果
3. **重试机制**：API调用失败时自动重试
4. **限流控制**：控制QPS，避免触发限流

### 7.3 Milvus优化

1. **批量插入**：积累一定数量后批量插入，提高效率
2. **索引优化**：根据数据量调整HNSW参数
3. **内存管理**：定期flush，确保数据持久化
